# 🔩 QC Fastener — Контроль качества крепежа

PWA-приложение для смартфона: сканирует Акт приёмочного контроля (рукописный или печатный), определяет позиции с отклонениями и автоматически формирует **Акт о несоответствии продукции** в формате DOCX по шаблону ООО «ОПМ».

---

## ✨ Функции

| Функция | Описание |
|---|---|
| 📷 **Сканирование ИИ** | Фотографируете акт — Claude распознаёт поставщика, заказ, товары и отметки дефектов |
| 📐 **ISO 3269 выборка** | Автоматический расчёт объёма выборки по объёму партии (AQL 1.0, уровень II) |
| 🏷️ **Классификация дефектов** | 4 категории по ГОСТ/ISO 8992: геометрия, поверхность, маркировка, упаковка |
| 📷 **Фото дефектов** | До N фото на каждую позицию, встраиваются в DOCX |
| 📄 **Генерация DOCX** | Готовый акт по шаблону ООО «ОПМ» скачивается прямо на телефон |
| 💾 **История** | Сохранение в локальное хранилище телефона |
| 📲 **PWA** | Установка как приложение на домашний экран (iOS/Android) |

## 🚀 Деплой на GitHub Pages (5 минут)

### 1. Создайте репозиторий

```bash
git init
git add .
git commit -m "init: QC Fastener app"
```

Создайте репозиторий на GitHub (например `qc-fastener`), затем:

```bash
git remote add origin https://github.com/ВАШ_ЛОГИН/qc-fastener.git
git branch -M main
git push -u origin main
```

### 2. Включите GitHub Pages

1. Откройте репозиторий на github.com
2. **Settings → Pages**
3. Source: **Deploy from a branch → main → / (root)**
4. Нажмите **Save**

Через 1–2 минуты приложение будет доступно по адресу:
```
https://ВАШ_ЛОГИН.github.io/qc-fastener/
```

### 3. Откройте на смартфоне

Отсканируйте QR-код или откройте ссылку в браузере (Safari на iOS, Chrome на Android).

**Установка на домашний экран:**
- iOS: Safari → «Поделиться» → «На экран «Домой»»
- Android: Chrome → «⋮» → «Добавить на главный экран»

## 🏷️ Классификация дефектов крепежа

### Геометрия / размеры
- Диаметр резьбы вне допуска
- Шаг резьбы не соответствует
- Длина болта вне поля допуска
- Проходит непроходной калибр
- Не проходит проходной калибр
- Несоответствие размера под ключ

### Поверхность / покрытие
- Ржавчина / коррозия
- Задиры, царапины, трещины
- Отслоение/пузыри цинкового покрытия

### Маркировка
- Нет клейма производителя
- Неверный класс прочности
- Нечёткое/отсутствующее клеймо

### Упаковка / документация
- Повреждена упаковка
- Неверная фасовка (количество)
- Отсутствует сертификат / паспорт

## 📄 Шаблон Акта несоответствия

Используется шаблон ООО «ОПМ» (ИНН 7802540772). Сформированный DOCX включает:
- Реквизиты сторон и поставки
- Таблицу позиций с отклонениями
- Классификацию дефектов (галочки в заключении)
- Данные выборки ISO 3269
- Фотоматериалы
- Подписи комиссии
